LinkedIn Markdown Formatter vs md2rich：隐私、功能、实测对比

2026 年 6 月 9 日上线了一个新工具 LinkedIn Markdown Formatter（markdowntolinkedin.com）：把 Markdown 粘贴到左边的框，右边实时显示 LinkedIn 风格的富文本预览，一键复制粘贴到 LinkedIn。它几乎和 md2rich 做同样的事——而且是同一天在社交媒体上被发现和讨论的。

这篇文章不是"甲更好"的结论，而是一个真实的对比。两个工具都刚刚上线或刚被关注，功能还在快速迭代。我花了一天时间在两个工具之间来回编辑、粘贴、发布、观察效果。以下是完整的对比报告。

1. 两个工具做什么

LinkedIn Markdown Formatter：打开 markdowntolinkedin.com，页面是左右分栏。左侧输入 Markdown，右侧渲染出 LinkedIn 风格的富文本。底部有个"Copy to Clipboard"按钮，点击后你可以直接 Cmd+V 粘贴到 LinkedIn 的帖子发布框或文章中。

这个工具在 2026 年 6 月 9 日上线，当天就在 Product Hunt 和社交媒体上被大量转发。它的定位非常精准：只做 LinkedIn 的 Markdown 转换。没有多余功能，没有广告，没有注册要求。

md2rich：2026 年初上线，也是左右分栏、实时预览、一键复制富文本。不同之处在于 md2rich 的定位更广：它不只为 LinkedIn 设计，而是适配 X Articles、Medium、Notion、WordPress 等多个平台的富文本格式。md2rich 也是 100% 纯前端、开源、零追踪。

3. 实测：同样的 Markdown，不同的输出

我用同一段 Markdown 分别在两个工具中渲染，复制后粘贴到 LinkedIn 的文章编辑器中，观察格式保留情况。

测试 Markdown

测试结果

LinkedIn Markdown Formatter 的输出：H1 渲染为 LinkedIn 的 Title 大小（约 28px）。H2 为 Section Header（约 22px）。列表缩进合适。代码块用灰色背景。引用用左边框+灰色斜体。整体感觉更像 LinkedIn 原生的"文章"编辑器风格。唯一的不足是表格支持不完整——用 Markdown 表格语法时，某些单元格的内容会错位。

md2rich 的输出：H1 略大（适合更宽的阅读场景）。H2 同样清晰。列表和代码块的表现类似。表格完整渲染且对齐正确。额外的优势是 md2rich 的渲染在跨平台时更一致——同样的内容粘贴到 X Articles 和 Medium 后排版不散架。

关键发现：两个工具的 LinkedIn 粘贴效果非常接近。非技术内容的创作者可能看不出明显区别。区别出现在（1）表格，（2）代码语法高亮，（3）需要跨平台发布的场景。

4. 适合什么人用哪个

选 LinkedIn Markdown Formatter 如果你：

  • 只使用 LinkedIn 一个发布平台
  • 只需要基本格式（标题、加粗、列表、引用）
  • 偏好极简界面（打开就编辑，没有多余选项）

选 md2rich 如果你：

  • 需要跨多个平台发布（LinkedIn + X Articles + Medium + Notion）
  • 文章中包含表格或代码块（md2rich 的表格渲染更完整）
  • 想要一个开源工具（代码可见，没有隐藏的数据处理逻辑）
  • 需要编辑区可调节大小、语法提示等便利功能

5. 隐私与安全对比

两个工具都强调"纯前端"。这是 Markdown 转富文本工具最安全的工作方式：

  • 你的 Markdown 内容只在浏览器内存中渲染
  • 没有数据发送到任何服务器
  • 关闭页面后内容消失

md2rich 更进一步：代码在 GitHub 上完全开源，任何人都可以审查代码确认没有隐藏的数据收集或信标。同时 md2rich 使用 Content Security Policy 头阻止任何外部脚本的加载——即使开发者想加入分析代码，浏览器也会阻止它。

LinkedIn Markdown Formatter 的源代码不可见。从网络行为看，它也没有后台请求（DevTools Network 面板干净），可以认为它是诚实的纯前端工具。但缺乏代码审计的可能性，对于内容安全要求极高的用户（如商业机密文档的作者），这是一个权衡点。

6. 结语

LinkedIn Markdown Formatter 的上线是好消息。它验证了"Markdown → 富文本"这个需求是真实且广泛的——不是一个工具在做，而是多个独立开发者同时看到了这个缺口。

两个工具各有侧重。LinkedIn Markdown Formatter 专注 LinkedIn 场景，做一件事做好。md2rich 覆盖更广的平台范围，同时提供更丰富的功能和更高的透明度。

我的建议：两个都收藏。如果你只在 LinkedIn 上发布基础格式的内容，用 LinkedIn Markdown Formatter。如果你需要更完整的格式支持、跨平台发布、或者希望使用开源工具，用 md2rich.com。两者都可以免费使用，不需要注册，你的数据安全。
