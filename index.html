Frequently Asked Questions

What is md2rich?

md2rich is a free online tool that converts Markdown to rich text format. Paste your Markdown, pick a platform (LinkedIn, Medium, Notion, X Articles, or plain HTML), and copy ready-to-publish rich text. Everything runs in your browser — no server, no upload, no signup.

Is md2rich really free?

Yes, md2rich is completely free. There are no limits, no hidden charges, no premium tiers. You paste Markdown and copy rich text — as many times as you want.

Does md2rich upload my Markdown to a server?

No. md2rich runs 100% client-side in your browser. Your Markdown never leaves your device. There are no backend servers, no database storage, and no tracking. You can verify this by opening your browser's DevTools Network tab.

Which platforms does md2rich support?

md2rich supports LinkedIn, Medium, Notion, X Articles (X Premium long-form posts), and plain HTML for any rich text editor. Each platform mode optimizes the output formatting for that specific platform's editor.

Do I need to create an account to use md2rich?

No account, signup, or login is required. Just load the page, paste your Markdown, and copy the rich text output.

Can I use md2rich offline?

Yes. Once md2rich has loaded in your browser, you can disconnect from the internet and it will continue working. All processing is done client-side with no server dependency.

How do I convert Markdown for LinkedIn with md2rich?

Select "LinkedIn (Markdown hint)" from the platform dropdown, paste or write your Markdown in the editor, then click "Copy as Rich Text". Paste directly into a LinkedIn post or article editor and the formatting — headings, bold, italic, lists, code blocks — will be preserved.
